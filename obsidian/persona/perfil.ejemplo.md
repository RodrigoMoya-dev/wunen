## Stack Tecnológico

| Tecnología | Nivel | Años de experiencia | ¿Dispuesto a trabajar con ella? |
|---|---|---|---|
| PHP | Avanzado | 10+ | Sí |
| WordPress / WooCommerce | Avanzado | 8+ | Sí |
| Laravel | Intermedio | 4 | Sí |
| JavaScript / VueJS | Intermedio | 5 | Sí |
| Docker | Intermedio | 3 | Sí |
| Python | Básico | 1 | Sí, si hay curva de aprendizaje razonable |
| Java / Spring Boot | Intermedio | 5 | Solo si el proyecto es atractivo |
| Odoo | Intermedio | 3 | Sí |
| React / Next.js | Básico | < 1 | Dispuesto a aprender |

**Tecnologías que prefiero evitar:**
- .NET / C# (sin experiencia, curva alta)
- Angular (prefiero Vue o React)
- Proyectos solo de mantenimiento legacy sin posibilidad de mejora

---

## Modalidad de Trabajo

- **Preferencia:** 100% remoto
- **Aceptable:** Híbrido (máx. 2 días presencial/semana) si la empresa está en Santiago, Chile
- **No acepto:** Presencial tiempo completo
- **Zona horaria:** Chile (UTC-3 / UTC-4 según horario de verano). Flexible ±3 horas para reuniones con Europa o EEUU

---

## Disponibilidad Geográfica

- Trabajo para empresas de cualquier país en modalidad remota
- Países preferidos para contratos: España, Chile, EEUU, México, Argentina
- No requiero visa para trabajar: Chile (ciudadano). Para España/UE necesitaría gestión de visa/contrato.

## Expectativa Salarial

| Moneda | Mínimo aceptable | Rango preferido | Ideal |
|---|---|---|---|
| USD (freelance/hora) | 20 USD/h | 25–35 USD/h | 40+ USD/h |
| EUR (freelance/hora) | 20 €/h | 25–35 €/h | 40+ €/h |
| CLP (relación de dependencia mensual) | $2.500.000 | $3.000.000–$3.500.000 | $4.000.000+ |

**Notas:**
- Para proyectos de largo plazo (+6 meses) puedo negociar hacia abajo si hay otros beneficios (flexibilidad, stack interesante).
- Para proyectos cortos o de alta especialización, el piso sube.

---

## Tipo de Trabajo Preferido

**Sí me interesa:**
- Proyectos de automatización de procesos
- Integraciones entre sistemas (ERP, CRM, APIs externas)
- Desarrollo de plugins/extensiones WordPress con lógica compleja
- Liderazgo técnico en equipos pequeños (2–5 personas)
- Consultoría Odoo (implementación, customización)
- Proyectos con uso de IA/LLMs (Claude, OpenAI, etc.)

**Me es indiferente:**
- Startups vs empresas establecidas
- Producto propio vs agencia (depende del proyecto)

**Prefiero evitar:**
- Soporte nivel 1 / helpdesk puro
- Proyectos de solo maquetado/diseño web sin desarrollo
- Empresas con stack muy legacy sin plan de modernización

## Sectores de Interés

**Preferidos:**
- Tecnología / SaaS
- E-commerce
- Automatización industrial o logística
- Sector agrícola / Agritech (experiencia previa en Ministerio de Agricultura)
- Joyería / artesanía (experiencia reciente)

**Excluidos:**
- Apuestas / casinos online
- Contenido adulto
- Empresas con malas referencias públicas de pago a freelancers

## Otros Criterios

- **Tipo de contrato:** Freelance (boleta de honorarios) o relación de dependencia. Ambos aceptables.
- **Duración mínima de proyecto:** Prefiero proyectos de al menos 1 mes. Proyectos de un día/semana solo si el valor hora compensa.
- **Proceso de selección:** Acepto pruebas técnicas de hasta 2 horas no remuneradas. Más de eso solo si son remuneradas.
- **Herramientas de comunicación:** Slack, Teams, Discord, email. Sin preferencia.
- **Metodología:** Ágil/Scrum es bienvenido. No es requisito.
